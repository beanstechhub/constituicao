Política de Privacidade

Última atualização: 27 de maio de 2026

1. Dados que coletamos

Nenhum dado pessoal é armazenado. O constituicao.tech opera de forma stateless:

  • Textos enviados para análise são processados em memória e descartados imediatamente após a resposta.
  • Arquivos (PDF, DOCX, TXT) são processados em memória e nunca persistidos em disco ou banco de dados.
  • Nenhum cookie é utilizado.
  • Nenhum rastreador (analytics, pixel, fingerprint) é empregado.

2. Endereço IP

Para fins de rate limiting (proteção contra abuso), o endereço IP do requisitante é convertido em hash irreversível (SHA-256 com salt). O IP original nunca é armazenado ou registrado em logs.

3. Logs

Os logs do sistema registram apenas:

  • Timestamp da requisição
  • Hash do IP (não reversível)
  • Tamanho do texto/arquivo (em caracteres/bytes)
  • Status HTTP da resposta
  • Request ID (UUID aleatório para suporte)

O conteúdo do documento nunca aparece nos logs.

4. Compartilhamento

Não compartilhamos dados com terceiros. Não vendemos, transferimos ou divulgamos informações de uso.

5. Base legal (LGPD)

Por não armazenar dados pessoais, o tratamento se limita ao necessário para execução do serviço (Art. 7, V, LGPD). Não há formação de perfil, decisão automatizada sobre pessoas, ou transferência internacional de dados.

6. Seus direitos

Como não armazenamos dados pessoais identificáveis, não há dados para acessar, corrigir ou excluir. Se tiver dúvidas, entre em contato via GitHub Issues.

7. Segurança

  • Comunicação via HTTPS (TLS 1.3)
  • Headers de segurança: HSTS, CSP, X-Frame-Options
  • Código aberto: auditável por qualquer pessoa

8. Contato

Dúvidas sobre privacidade: GitHub Issues
