Termos de Uso

Última atualização: 27 de maio de 2026

1. Aceitação

Ao usar o constituicao.tech (site, API ou GitHub Action), você concorda com estes termos.

2. Descrição do serviço

O constituicao.tech é uma ferramenta de análise estatística e determinística de documentos. Fornece indicadores de probabilidade, não vereditos definitivos. A decisão final é sempre humana.

3. Uso gratuito e limites

  • API pública: 20 análises por dia por IP
  • Burst protection: 5 requisições por minuto por IP
  • Tamanho máximo: 10MB por arquivo, 500.000 caracteres por texto
  • Uso self-hosted: sem limites (sujeito à licença Apache 2.0)

4. Uso permitido

Você pode usar o serviço para:

  • Verificação de integridade de documentos próprios
  • Integração em pipelines de CI/CD
  • Pesquisa e educação sobre segurança de IA
  • Qualquer uso compatível com a licença Apache 2.0

5. Uso proibido

  • Uso para discriminação ou negação de direitos fundamentais
  • Bloqueio automático de acesso à justiça baseado exclusivamente no resultado
  • Tentativa de sobrecarregar o serviço (DoS)
  • Engenharia reversa para construir ferramentas de ataque

6. Isenção de responsabilidade

O serviço é fornecido "como está" (AS IS), sem garantias de qualquer tipo. Os resultados são indicativos e estatísticos. Não substituem:

  • Análise pericial humana
  • Verificação em certificadora oficial (ICP-Brasil)
  • Decisão judicial fundamentada
  • Avaliação docente

Beans Tech não se responsabiliza por decisões tomadas exclusivamente com base nos resultados da ferramenta.

7. Disponibilidade

Não garantimos disponibilidade 24/7. O serviço pode ser suspenso para manutenção sem aviso prévio. Para uso crítico, recomendamos deploy self-hosted.
